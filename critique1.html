Layout

The layout of Abduzeedo.com is very simple and easy to navigate. Since it is an image driven site with images for every content element, the layout is consistent throughout, which consists of an image, heading, byline and text. It uses a flexible grid with 4 or 5 columns depending on the user's browser window size.

Color

The color scheme of the site is very simple and muted. It primarily uses black, gray and white. The simple color scheme for the site is excellent because it is an image driven site. The only colors used other than the blacks and grays are the colors of the images and a very colorful logo in the header.

Design

The design of the site uses the standard of Web design today. It has a simple 3D navigation bar with 3D rollover buttons. The logo has a very colorful rainbow design with an outer glow. Those are the main design elements throughout. All of the copy uses sans-serif fonts, and the headings create a good hierarchy.

Navigation

Abduzeedo.com has great navigation. There is a primary navigation bar at the top of every page, which contains links to Tutorials, Inspiration, Wallpapers and Interviews as the dominant links. Abduzeedo.com takes an interesting approach to secondary navigation by including things that would regularly be included in the footer on the global navigation bar. The navigation also includes links to social networking sites, RSS feeds and search engines.

Animated Ads

In this tutorial we are going to teach you how to create a simple 3D text technique with masks in Photoshop that can be used for a variety of designs to add more depth to your creations.


Step 1:

Pick a texture that you would like to use that matches the mood of your project, and open it. For this tutorial I am going to use a stone texture. Duplicate the texture and turn the bottom one off.

Step 2:

Choose some fonts to use. San-serif fonts work the best for this technique, but any font can be used with slight adjustments in the Character panel. For this tutorial I am going to use a simple bold Verdana font

Step 3:

Create your text the way you want it to look.

Step 4:

Right click the thumbnail next to the text in the Layers palette and click Select Pixels.

Step 5:

With the text layer selected and the pixels selected, click Select on the main menu and select Inverse.

Step 6:

Once you have done that, turn off the text layer, select the image you are going to engrave the text into and click the Add Layer Mask icon in the Layers palette.

Step 7:

Turn on the bottom image that is the duplicate of the one you engraved the text into. Alter it however you want to get the desired effect. For this example I only adjusted the Levels under Image / Adjustments.

Step 8:

Select the layer with the mask and add a Drop Shadow Layer Style. Keep the Distance, Spread and Size low for the most desirable effect. You are then finished. For some more fun, use the brush tool to add certain effects to the mask.
